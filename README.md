# LatAm Investment Research Agent

An investment research agent for emerging markets in Latin America

uv sync --extra dev

uv run python -m latam_investment_research_agent.main



uv run latam_investment_research_agent

uv run pytest

uv run ruff check .
uv run ruff format .
